PINTEREST

My first UI/UX design project created entirely in Figma where I recreated and redesigned multiple Pinterest mobile application screens. Through this project, I explored mobile design systems, visual hierarchy, spacing, reusable components, interface consistency, and learned how modern consumer applications structure intuitive user experiences.

Role: UI/UX Designer Timeline: First Figma Project Figma • Mobile UI • Prototyping

Designed Screens

What I Designed

Login & Onboarding

Designed authentication screens focusing on user entry flow, account creation patterns and familiar app interactions.

Home Feed Layout

Recreated Pinterest-style masonry feed understanding content hierarchy and image-first browsing patterns.

Search Experience

Designed discovery screens helping users explore content ideas and browse inspiration collections.

Boards Interface

Created board management layouts helping users save and organize collections through intuitive interactions.

Profile Screen

Designed user profile pages showing boards, pins, saved content and social interaction structure.

Settings Navigation

Learned app architecture by recreating settings pages, account management and navigation systems.

What This Project Taught Me

This project marked the beginning of my UI/UX journey. It helped me understand how real-world applications are structured, how spacing and component systems create consistency, and how thoughtful interface design shapes seamless digital experiences. It became the foundation for my later product design work.
